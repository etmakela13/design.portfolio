People Helping People

Web Development 302: User Interface Design and Development

Spring 2023

Role Played: I was one of three UX designers working on the project. We all had equal say and collaborated on each decision.

Project Description: Create a mock site for a non-profit volunteering company to streamline business and communication between employees and participants.

Different Deliverables: UI Components, Grids/Artboards, Palette, Typography, Icons, Fonts, Personas, Low Fidelity Wireframe, High Fidelity Wireframe, Site Map.


Initial Designs for Pages





Design Elements



Our project required a full glossary of all the different UI components that were used within the site. These are a couple of the components that we used and how we used them.





Finalized Designs



Our final design was an interactive html, javascript, and css page that covered the basics of the site. This included the login, registration, and home pages. As well as the search feature, the focused event card, and the request for help card.



Results

Final Product: Overall, the design ended up being something that our group was happy with. The only thing we didn't like was that we didn't get to finish the chat page before our deadline.

Feedback: This was a fictional site so the feedback received was from classmates and professors. Along with a couple smaller inconsequential things with the site, the big issues brought forward was the lack of color differences therefore, users with color blindness may have trouble distinguishing the different colors.
